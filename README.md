# Customer-shopping-behavior-analysis-
Customer Shopping Behavior Analysis
Overview

This project analyzes customer shopping behavior to uncover insights related to purchasing patterns, customer segmentation, revenue trends, subscription impact, and shopping preferences using Python, PostgreSQL, and Power BI.

The project demonstrates a complete end-to-end data analytics workflow:

Data Cleaning & Feature Engineering
SQL Business Analysis
Interactive Dashboard Creation
Business Insight Generation
